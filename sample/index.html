Sample app for angular-breadcrumb

This app is a basic information system for hotel. Its main purpose is to illustrate how work the module angular-breadcrumb in various situations.

Here is the state machine : 

─ home (defined as prefix in the breadcrumb)
─ booking
    └── booking.day (use the angular date filter in label)
          └── booking.day.detail (open a popup, not displayed in breadcrumb)
─ room
    └── room.detail (view included in root's view instead of parent's one, variable in breadcrumb label)
          └── room.detail.edit (allow to see the working parameterized link for parent 'room.detail')
